Ovaries

Release eggs and make hormones like estrogen and progesterone.

Uterus

Builds a lining each month. If there’s no pregnancy, the lining sheds — that’s a period.

Cycle basics

Ovulation usually happens mid-cycle. Cycle length and flow vary by person.
